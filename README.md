# ECDOsimulator

**Exploring Exothermic Core-Mantle Decoupling Dzhanibekov Oscillation (ECDO) Theory through open-source data and simulations.**

ECDO is an emerging theory concerning potential Earth core-mantle dynamics with implications for geophysics, planetary science and humanity. This repo aims to serve as the open-source hub for progress towards a more perfect Earth digital twin for unlocking improved ECDO simulations and resilience planning.
